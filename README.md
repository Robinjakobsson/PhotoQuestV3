## Run Locally

Clone the project

```bash
  git clone https://github.com/Robinjakobsson/PhotoQuestV3.git
```

Go to the project directory

```bash
  Add the firebase json
```
For Google and Facebook auth, you need to connect your app with your dev account for a key.
